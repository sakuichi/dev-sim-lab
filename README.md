# dev-sim-lab

ソフトウェア開発の誤解をインタラクティブに可視化する **One Issue = One Webページ** シミュレーターシリーズ。

- 公開先: https://sakuichi.github.io/dev-sim-lab/ (予定)
- 起点: [QCDトレードオフシミュレーター](https://sakuichi.github.io/qcd-tradeoff/)(本リポジトリに移設予定)
- 対象: 経営層・ステークホルダー向けの教育・トレーニング素材
- 言語: 日本語・英語(i18n対応)

## 要件定義

詳細は [docs/requirements.md](docs/requirements.md) を参照。

## 技術スタック
